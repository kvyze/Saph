# Установка
Для установки Saph можно воспользоваться двумя разными способами.

### Установка через SaphSetup
1. Скачайте _SaphSetup.exe_ из вкладки **Releases**;
2. Откройте файл и, пройдя все пункты, завершите установку;
3. Проверьте статуc установки, введя команду ```saph -v``` в консоль. Если вы увидели что-то по типу "Saph 0.1.0 win64",
то установка прошла успешно.

### Ручная установка
1. Скачайте архив _Saph-<версия>.zip_ из вкладки **Releases**;
2. Распакуйте его в любую удобную для вас папку;
3. Добавьте папку с языком в **PATH** для удобного использования.

# Запуск
Чтобы запустить Saph-скрипт, создайте файл с расширением .sa (например, _main.sa_).
Напишите туда что-то наподобие:
```
print (437 + 279) * 27 - 13
```
Откройте консоль в папке с файлом и напишите следующую команду:
```
saph main.sa
```
Если вы получили 19319, то всё работает, если нет, скорее всего, я снова накосячил в релизе, ждите обновления.

# Сборка в .exe
Чтобы собрать код Saph в исполняемый файл, установите на свой ПК **gcc**
(скачать можно либо с [официального сайта](https://gcc.gnu.org/install/download.html), либо через [MSYS2](https://www.msys2.org/)).
Откройте консоль в папке с файлом Saph и введите следующую команду:
```
saph main.sa -b
```
После этого в той же папке появится .exe файл. С другими командами вы можете ознакомиться, введя в консоль:
```
saph -h
```
